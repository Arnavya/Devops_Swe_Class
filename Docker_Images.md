## Q1. Pulling an Image

```bash
docker pull nginx:1.23.4 > answer.txt

# Verify the image is downloaded
docker images

# Verify the logs
cat answer.txt
```

- `>` : Redirects the output of the command to a file
- `>>` : Appends the output of the command to a file
